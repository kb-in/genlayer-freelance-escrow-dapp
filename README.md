# GenLayer Freelance Escrow dApp

A simple AI-powered escrow contract built on GenLayer.

## What it does

- Client posts a job and locks payment
- Freelancer submits work
- AI validators evaluate the work
- Payment is released or refunded automatically

## Tech used

- GenLayer Intelligent Contracts (Python)
- Optimistic Democracy consensus
- Equivalence Principle (strict equality)
